# Images des catégories du catalogue

Ce dossier contient les images de produits phares pour chaque catégorie affichée sur la page `/catalogue`.

## Images requises

Chaque image doit être au format PNG avec fond transparent (détouré). Format recommandé : 800x800px minimum.

### Liste des images nécessaires :

1. **outils-electroportatifs.png** - Exemple : Perceuse M18 FUEL
2. **batteries-chargeurs.png** - Exemple : Batterie M18 REDLITHIUM HIGH OUTPUT
3. **exterieurs-espaces-verts.png** - Exemple : Tronçonneuse M18 FUEL
4. **assainissement.png** - Exemple : Déboucheur M18
5. **eclairage.png** - Exemple : Projecteur ROVER M18
6. **instruments-mesure.png** - Exemple : Laser rotatif
7. **aspirateurs.png** - Exemple : Aspirateur M18 FUEL
8. **epi.png** - Exemple : Chaussures de sécurité Milwaukee
9. **vetements.png** - Exemple : Veste chauffante M12

## Fallback

Si une image n'est pas disponible, le composant affichera automatiquement une lettre initiale (première lettre du titre de la catégorie) sur fond rouge Milwaukee.

## Conseils visuels

- Utilisez des photos détourées haute qualité (sans arrière-plan)
- Choisissez des produits iconiques et facilement reconnaissables
- Privilégiez les angles 3/4 qui montrent bien le produit
- Les images doivent être optimisées pour le web (compression sans perte de qualité)
- Format recommandé : PNG avec transparence, 800x800px à 1200x1200px
